## Overview
This project implements a linear/integer programming optimization model to support warehouse order fulfillment and production planning decisions. The model maximizes total profit subject to inventory availability, labor-hour, and capacity constraints.

## Tools Used
- Excel Solver
- VBA (UserForms, Solver automation)
- Linear / Integer Programming

## Features
- Interactive UserForm to vary operational constraints
- Automated Solver execution via VBA
- Scenario and bottleneck analysis through constraint variation

## How to Run
1. Download the Excel file
2. Enable macros
3. Click "Run Optimization"
4. Adjust inputs in the UserForm and solve
